# DisasterNet
This repository Contains the material of DisasterNet project. It has the preprocessed Dataset, the source files for preprocessing , Keras code for implementation of the model.

# Dataset
The dataset is divided into 70% training and 30% testing dataset. The images size is set to 100x100. The original dataset is collected by google search by Gautam Kumar from Sardar Vallabhbhai National Institute of Technology, Surat. As collected via google search the images size were not same. We have done the preprocessing. The Script for preprocessing is also added in the repository.
